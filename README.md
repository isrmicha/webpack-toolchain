# Webpack toolchain - Desenvolvimento web vanilla

- Para inicializar
  - **NPM**

    ```bash
    npm i
    ```

  - **YARN**
  
    ```bash
    yarn
    ```

- Para rodar webpack dev server com hot reload
  - **NPM**

  ```bash
  npm run dev
  ```
  
  - **YARN**

  ```bash
  yarn dev
  ```

- Para buildar

  - **NPM**

    ```bash
      npm run build
    ```

  - **YARN**

    ```bash
      yarn build
    ```

- Para subir servidor express para servir o conteudo buildado

  - **NPM**

    ```bash
      npm run server
    ```

  - **YARN**

    ```bash
      yarn server
    ```
